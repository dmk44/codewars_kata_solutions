# Решения задач с Codewars

[EN](readme.md)

Это мой личный репозиторий с решениями kata разных уровней с **Codewars**.

Я использую его как базу знаний: отслеживать прогресс, возвращаться к приёмам и идеям в разных языках, держать под рукой удобные примеры.
Ну и заодно какие-то стандартные команды git не забывать :-D

### Языки, которые я изучаю:
- Python
- R
- JavaScript
- Java
- C++
- C#
- Bash
- PowerShell
- SQL


Мне нравится пробовать разные языки — не только ради узкой специализации, но и для расширения кругозора и понимания разных парадигм.
А CodeWars и подобные ресурсы - отличная практика.

**Важно:**  
Codewars официально не рекомендует публиковать такие репозитории.  
Если представители Codewars обратятся ко мне, репозиторий будет удалён или доступ будет ограничен.
